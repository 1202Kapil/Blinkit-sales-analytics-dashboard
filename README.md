# 🛒 Blinkit Sales Performance Dashboard
1. Project Title / Headline

Blinkit Sales Analytics Dashboard – Power BI

An interactive Power BI dashboard designed to analyze Blinkit's grocery sales performance across different outlet types, item categories, and locations to uncover key business insights.

2. Short Description / Purpose

The Blinkit Sales Dashboard provides a clear overview of sales performance, product demand, and outlet distribution. It helps identify high-performing product categories, outlet tiers, and location-based sales trends for better business decision-making.

3. Tech Stack

The dashboard was built using the following tools and technologies:

• 📊 Power BI Desktop – Used for building interactive dashboards and visualizations.
• 🔄 Power Query – Data cleaning and transformation.
• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures such as Total Sales, Average Sales, and Ratings.
• 🧩 Data Modeling – Structured dataset relationships to enable efficient filtering and analysis.
• 📁 File Formats – .pbix for development and .png for dashboard preview.

4. Data Source

Dataset: Blinkit Grocery Sales Data

The dataset includes information about grocery products sold through Blinkit, covering:

a) Item types and fat content

b) Outlet establishment year

c) Outlet size and location tier

d) Item visibility and ratings

e) Sales performance metrics

The dataset contains 8,523 items across multiple outlet types and locations.

5. Features / Highlights
• Business Problem

Blinkit operates across multiple outlet types and locations. Understanding which product categories, outlet sizes, and regions drive the most sales is essential for improving inventory planning and sales strategy.

• Goal of the Dashboard

The dashboard aims to:

a) Monitor overall sales performance

b) Identify top-performing product categories

c) Compare sales across outlet tiers and sizes

d) Analyze customer ratings and product visibility

• Business Impact & Insights

📈 Sales Optimization – Identify high-demand product categories.
🏪 Outlet Strategy – Understand which outlet sizes and tiers perform best.
📦 Inventory Planning – Improve stock management based on item demand.
📍 Market Expansion – Identify potential regions for business growth.
